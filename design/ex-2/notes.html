Exercise 2: Basics of Colors

Challenges Faced:

Diving into color theory felt overwhelming initially—balancing contrast and visual harmony was trickier than I thought. My first attempt at a monochromatic theme looked bland, while complementary schemes often clashed too much. Also, switching between dark and light themes using CSS variables was challenging because small tweaks would disrupt the overall design.

Tools and Resources Used:

  • HTML5 for structure
  • CSS Variables for dynamic color theming
  • HSL values for fine-tuning color shades and saturation
  • Developer Tools to test color contrast and readability
  • Media Queries for responsive color palettes

My Thought Process:

I started by experimenting with a monochromatic scheme—using a single base color and playing with opacity to make the card stand out. Then I moved to a complementary color palette, trying to strike a balance between contrast and aesthetics. Adding an accent color for buttons and headers gave the page a nice finishing touch.

For Level 2, I set up CSS variables to manage dark and light themes. Switching between them exposed issues with text contrast over images, so I had to fine-tune overlays for better readability. Generating a color swatch grid with HSL was fun but required careful adjustments to maintain consistency.

Technical Approaches:

  • Used CSS variables for flexible theming.
  • Leveraged the HSL color model for smooth adjustments in lightness and saturation.
  • Implemented media queries for responsive color palettes.
  • Experimented with gradients for Level 3—linear for headers, radial for buttons.

Key Learning Outcomes:

  • How to create and balance monochromatic, complementary, and accent color schemes.
  • Practical use of CSS variables to toggle between dark and light themes.
  • Improved understanding of color contrast and accessibility.
  • Mastered basic gradient techniques with pure CSS.

Specific Challenges:

  • Ensuring color contrast met accessibility standards.
  • Making themes switch smoothly without breaking the design.
  • Generating visually appealing color swatches with consistent saturation.

Potential Improvements:

  • Adding smoother transition effects for theme switching.
  • Testing designs with advanced accessibility tools.
  • Experimenting with more complex gradient combinations.

Reflection:

This exercise gave me a solid understanding of how color impacts web design. I realized that even small adjustments—like tweaking opacity or changing saturation—can drastically improve the visual experience. Using AI for inspiration helped me explore different color schemes when I felt stuck, but fine-tuning the details was all about trial and error. Overall, a great learning experience on combining creativity with usability!
