Thought Process for the Blog Post Exercise

Approach to the Problem

The exercise seemed simple enough at first. The goal was to create a blog post and style it using basic HTML and CSS. My approach was to break down the task into logical steps:

  • Start with a simple HTML structure, adding a title and subtitle to the blog post.
  • Then, work on styling the content using CSS, focusing on alignment and margins.
  • For the next level, I needed to add a list, a blockquote, and an inline code snippet, ensuring that each element was styled properly using CSS element selectors.
  • For the final step, I planned to use a classless CSS framework to enhance the design of the blog post, experimenting with different options like Classless.de to keep it minimalistic yet visually appealing.
